# AI-Health-Report-Simplifier-
Medical reports are often difficult for patients to understand due to technical terminology. AI Health Report Simplifier bridges this gap by converting medical data into simple language, identifying potential health risks, and offering guidance for informed decision-making—empowering patients while supporting, not replacing, doctors.
# 🏥 AI Health Report Simplifier PRO

An AI-powered healthcare assistant that analyzes medical reports and converts complex medical terminology into simple language.

## 🚀 Features

✔ Upload **PDF medical reports**  
✔ Upload **image reports using OCR**  
✔ Convert **medical terms to simple language**  
✔ **Hindi and Marathi translation**  
✔ **Disease risk prediction**  
✔ **Smart medical advice**  
✔ **Patient profile + BMI analysis**

## 🧠 Technologies Used

- Python
- Streamlit
- OCR (Tesseract)
- PDF Processing
- Natural Language Simplification
